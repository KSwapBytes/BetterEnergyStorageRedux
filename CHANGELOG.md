# Changelog

## 0.3.0

- Rebuilt Better Energy Storage Redux as a faithful content-only port using vanilla Power Storage behavior.
- Added upgraded tiers Mk1 through Mk6.
- Set unlock tiers to Mk1 Tier 2, Mk2 Tier 3, Mk3 Tier 4, Mk4 Tier 5, Mk5 Tier 7, and Mk6 Tier 8.
- Preserved vanilla-derived icons, meshes, storage behavior, save/load behavior, and power grid compatibility.
- Known limitation: mixed battery tiers on the same power grid may not preserve individual authored charge-rate caps because Satisfactory handles batteries as a grid-level storage group.
